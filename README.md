## Build WINE_LG for PortProton

Create chroot debian:
```# 01_create_debian_multi.sh```

Get source WINE_TKG and patchs it:
```$ ./02_get_wine tkg```

Build from sources:
```$ 03_build_wine_debian_multi.sh```
